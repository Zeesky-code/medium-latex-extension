# Medium LaTeX to Unicode Extension

A Chrome Extension that allows you to write LaTeX and convert it to Unicode for easy pasting into Medium (and other rich text editors).

## Features
- **Live Preview**: See the Unicode result as you type.
- **LaTeX Support**: Greek letters, arrows, operators, superscripts (`^2`), and subscripts (`_i`).
- **Complex Subscripts**: Supports grouped subscripts like `x_{i,j}` -> `xᵢ,ⱼ`.
- **Visual Symbol Picker**: Clickable grid of common symbols.
- **History**: Remembers your recently copied snippets.
- **Smart Styles**: Bold (`\mathbf`) and Monospace (`\mathtt`) support.

## How to Install (Load Locally)

1.  Open Google Chrome.
2.  Navigate to `chrome://extensions`.
3.  Enable **"Developer mode"** in the top right corner.
4.  Click the **"Load unpacked"** button in the top left.
5.  Select the project folder:
    `medium-latex-extension`
6.  The extension icon (green Sigma) should appear in your toolbar.

## Usage
1.  Click the extension icon.
2.  Type your LaTeX (e.g., `\alpha \to \beta`).
3.  Click "Copy to Clipboard".
4.  Paste into Medium!
